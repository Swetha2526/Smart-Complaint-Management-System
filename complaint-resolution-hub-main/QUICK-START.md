# 🚀 Quick Start Guide - Complaint Resolution Hub

## ⚡ Fastest Way to Run the App

### Windows
1. Double-click: `start-dev.bat`
2. Wait for terminal to show: `Local: http://localhost:5173/`
3. Browser opens automatically

### Mac/Linux
```bash
chmod +x start-dev.sh
./start-dev.sh
```

### Manual Method (Any OS)
```bash
npm install        # (if not done yet)
npm run dev
```

Then visit: **http://localhost:5173/**

## 🐛 Quick Troubleshooting

**Dev server won't start?**
```bash
npm install
npm run dev
```

**Port 5173 in use?**
Edit `vite.config.ts` to change port, or kill process using port 5173

**Styles not loading?**
```bash
npm run build
npm run preview
```
